STEPS FOR LINKING LOCAL FOLDER WITH GITHUB

git init --  to link with github
git add . -- add all files into github
git commit -m "comment" -- give update about code
git branch -M main -- add branch
git remote add origin https://github.com/alijunaid786/simple-calculator.git -- remotely add code into github profile
git push -u origin main -- push code into github

STEPS WHEN FURTHER CHANGES IN CODE

git add .
git commit -m "commit (what changes has been made)"
git push origin main
